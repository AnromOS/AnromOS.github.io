Windows系统中，使用刷机工具进行刷机

1.下载刷机工具

刷机工具下载地址
下载完成后，解压

2.下载ROM

根据机型，下载对应的ROM
一加手机一代
华为Nexus6P
Google Nexus 5x
Google Nexus 5
小米手机Mi3，Mi4
一加手机3

3.ROM改名

下载完成后，将下载的zip文件改名为anromupdate.zip
并移动到刷机工具文件夹中，此时，刷机工具文件夹的内容如下图:

4.将手机重启到recovery

关机后，长按电源键+音量UP，进入recovery模式

点击Keep Read Only

点击Advanced

点击ADB Sideload

滑动Swipe to Start Sideload

5.运行刷机脚本

运行刷机脚本前，请彻底关闭电脑端的360手机助手，QQ应用宝等第三方手机助手软件
双击刷机文件夹中的anrom.bat
效果如图

等待刷机过程完成
当手机屏幕上出现如下的Successful信息时，点击Reboot System

至此，刷机过程就完成了

常见问题

  1. anrom.bat脚本闪退
    检查电脑上的第三方手机助手软件是否关闭
    重新插拔数据线
    换一根数据线
  2. 无法进入sideload状态
    回退到主菜单，并选择Reboot

    选择reboot到recovery

    从第4步开始，重新刷机
